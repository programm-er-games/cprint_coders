<h1>Система мониторинга и управления освещением в офисе</h1>

В корневой папке находятся файлы для запуска кода для ESP32. В папке cprint_coders_app находятся файлы для запуска веб-приложения для мониторинга и управления системой освещения.

<b>Для запуска приложения: </b>
<ol>
  <li>Распаковать node_modules в ту же папку, где он находится</li>
  <li>Запустить run.vbs</li>
</ol>

<b>Для запуска скетча:</b>
<ol>
  <li>Загрузить файлы из корневой папки (кроме папки с приложением) в проект Wokwi</li>
  <li>Запустить компиляцию кода</li>
</ol>

<b>Готово!</b>

Наш проект работает по следующему сценарию:
<ul>
  <li>ESP32 считывает данные с фоторезистора (освещённость), ультразвукового датчика ("наличие людей") и лампочки (яркость лампы)</li>
  <li>Все считанные данные ESP32 отправляет приложению при помощи веб сокетов (при условии, что и ESP32, и устройство подключены к одной сети WiFi)</li>
</ul>
